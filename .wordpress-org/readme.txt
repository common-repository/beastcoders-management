== Description ==

This Plugin is made to provide user the smallest basic functionalities. We have added just one functionality right now:
1) Disable Admin Bar

We will be adding much more functionalities as soon as possible. This will provide much higher flexibility to users.

== Installation ==

1. Upload `beastcoders` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu where `beastcoders Management` plugin is located in WordPress
