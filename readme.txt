== Description ==
This Plugin is made to provide user the smallest basic functionalities. We will be adding Widgets, Shortcodes and much more We have added functionalities right now:

1) Disable Admin Bar

2) Facebook Like Box Widget 

Future functionalities:

1) FAQ Manager

2) Likes Functionality ( Coming Soon )


We will be adding much more functionalities as soon as possible. This will provide much higher flexibility to users.

== Installation ==
1. Upload `beastcoders-management` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu where `beastcoders Management` plugin is located in WordPress
